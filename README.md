# 2026 跨年 & 生日庆典网页部署指南

这是一个基于 React 和 Tailwind CSS 的沉浸式庆典网页。

## 🌐 免费上线步骤

### 方案 A：使用 Vercel (推荐)
1. 将所有文件上传至你的 GitHub 仓库。
2. 登录 [Vercel](https://vercel.com/)。
3. 点击 **Add New Project** 并导入该 GitHub 仓库。
4. 部署环境会自动识别静态文件，点击 **Deploy** 即可完成。

### 方案 B：使用 GitHub Pages
1. 在 GitHub 仓库设置中找到 **Settings** -> **Pages**。
2. 在 **Build and deployment** 下选择 `Deploy from a branch`。
3. 选择 `main` 分支和 `/ (root)` 文件夹，点击 **Save**。
4. 等待几分钟，GitHub 会提供一个 `https://username.github.io/repo-name/` 的链接。

## 📱 访问提示
- **全屏体验**：在手机浏览器中打开后，建议点击“添加到主屏幕”以获得沉浸式体验。
- **声音互动**：进入庆祝阶段后，记得开启系统音量并对准麦克风轻轻吹气！
